# Portal Notícias — Takta no Ar

## Como rodar

### Pré-requisito
Ter o [Docker](https://www.docker.com/) instalado.

### Passo a passo

```bash
# 1. Entre na pasta do projeto
cd Portal_Noticias

# 2. Suba o container
docker compose up --build

# 3. Acesse no navegador
http://localhost:8081
```

### Para parar

```bash
docker compose down
```
